An example of facebook auth by devise + omniauth
====

Configuration
----

edit `config/initializers/1\_facebook.rb`

    FACEBOOK_CONFIG = {
      :app_id => 'APP_ID',
      :secret => 'APP_SECRET',
      :app_namespace => 'APP_NAMESPACE'
    }
